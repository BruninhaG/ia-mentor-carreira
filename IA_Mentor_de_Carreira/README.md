# IA Mentor de Carreira

Projeto desenvolvido como desafio de código nível Júnior.

## Objetivo
Auxiliar pessoas a descobrirem áreas da tecnologia compatíveis com seu perfil usando lógica de recomendação (IA baseada em regras).

## Tecnologias
- HTML
- CSS
- JavaScript

## Funcionalidades
- Coleta de perfil
- Sistema de recomendação inteligente
- Interface moderna

## Como executar
Abra o arquivo index.html no navegador.
